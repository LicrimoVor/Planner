# Планировщик задач

## Описание

Проект представляет собой веб-приложение, созданное с использованием Python и Django. Приложение позволяет пользователям создавать свои задачи, управлять ими и отслеживать их выполнение.

---

Ключевые технологии и фреймворки, используемые в этом проекте, включают:

**Python** : Язык программирования, используемый для внутренней части приложения.
**Django**: веб-платформа высокого уровня на Python, которая соответствует архитектурному шаблону model-view-controller (MVC). Она предоставляет структуру для создания веб-приложений и обрабатывает общие задачи, такие как аутентификация и управление базами данных.
**Django REST Framework (DRF)** - мощный и гибкий инструментарий для создания API на Django. Он предоставляет набор инструментов для обработки сериализации, аутентификации и маршрутизации в веб-приложениях.
**React**: библиотека JavaScript для создания пользовательских интерфейсов. Он используется во внешнем интерфейсе приложения для создания интерактивных и адаптивных компонентов пользовательского интерфейса.
**Redux**: Библиотека управления состоянием для приложений на JavaScript. Он используется во внешнем интерфейсе для управления состоянием приложения и обработки потока данных.
**PostgreSQL** - мощная объектно-реляционная система баз данных с открытым исходным кодом. Используется в качестве базы данных для хранения прикладных данных.
**Redis** - хранилище структуры данных в памяти, используемое для кэширования и постановки сообщений в очередь.
**Docker**: платформа контейнеризации, используемая для упаковки и развертывания приложения.
**Docker Compose**: инструмент для определения многоконтейнерных приложений Docker и управления ими.

---

Приложение работает по архитектуре клиент-сервер, где интерфейс отвечает за визуализацию компонентов пользовательского интерфейса и обработку взаимодействий с пользователем, в то время как серверная часть обрабатывает обработку данных, хранение и конечные точки API.

Проект организован в виде отдельных каталогов и модулей для различных компонентов, таких как модели, представления, сериализаторы и шаблоны. Он соответствует архитектурному шаблону MVC, где модели представляют структуру данных, представления отвечают за логику и обработку данных, а шаблоны - за уровень представления.

Чтобы запустить приложение, вам необходимо выполнить шаги по установке, упомянутые ранее. Доступ к приложению можно получить через веб-браузер по указанному URL-адресу.

В целом, этот проект предоставляет комплексное решение для управления задачами с такими функциями, как создание, редактирование, удаление, фильтрация и сортировка задач. Оно также включает аутентификацию пользователей, авторизацию и управление доступом на основе ролей. Приложение разработано таким образом, чтобы быть масштабируемым, поддерживаемым и простым в развертывании.
